Sobre o projeto

Uma solucao para transformar dados orbitais em alertas uteis para a populacao

O OrbitAlert apresenta uma experiencia responsiva para consultar regioes monitoradas, entender o nivel de risco e receber uma recomendacao de acao.

Problema abordado

Desastres e eventos climaticos podem atingir uma regiao antes que a populacao consiga se preparar. Quando a informacao chega tarde, moradores perdem tempo procurando rotas, pontos de apoio e orientacoes confiaveis.

Proposta

A plataforma simula o cruzamento de dados orbitais, localizacao e indicadores de risco para priorizar alertas, destacar areas afetadas e orientar a resposta de moradores e equipes de apoio.

Tecnologias

Base tecnica do Front-End

HTML semantico

Estrutura organizada com cabecalho, menu, conteudo principal, secoes e rodape.

CSS responsivo

Layout adaptavel com grid, flexbox, variaveis e media queries.

JavaScript puro

Menu mobile, abas, FAQ, modal e validacao de formulario sem bibliotecas externas.
